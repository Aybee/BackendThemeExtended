BackendThemeExtended

Some selectable options to pimp the default backend theme plus some improvements.

Description

Options
  • Fix buttons on top or bottom
  • Small previews in parent lists
  • Fix navigation
  • Fix header
  • Hide mouseover tooltips of clickable elements or move them to the top
  • Add an optional color theme
Features
  • new* Adjusts the ACE editor like [ace_extended] (but only 3 lines when empty)
  • Adjust scroll if neccessary while opening palettes
  • Higher area for clicking labels (uses unused space)
  • Some cool improvements with the optional color theme
  • Fixes a datepicker bug
Misc

Does not touch the iframe popups.

Please report bugs.
